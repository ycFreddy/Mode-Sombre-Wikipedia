## Pour essayer le rendu sur Firefox

Rendez vous sur la page addon de Firefox<br>**https://addons.mozilla.org/fr/firefox/addon/mode-sombre-pour-wikipedia/**

## Pour essayer le rendu sur Chrome

1. Téléchargez et décompressez le zip du repository
2. Dans chrome, ouvrez ce lien : **"chrome://extensions/"**
3. Passez en Mode développeur
4. cliquez sur **"Charger l'extension non empaquetée"** et sélectionnez le dossier décompressé
5. Allez sur Wikipedia

## Pour essayer le rendu sur Edge

1. Téléchargez et décompressez le zip du repository
2. Dans chrome, ouvrez ce lien : **"edge://extensions/"**
3. Passez en Mode développeur
4. Cliquez sue **"Chargez l'extension décompressée"** et sélectionnez le dossier décompressé
5. Allez sur Wikipedia
